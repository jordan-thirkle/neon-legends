# Neon Legends — Idle RPG

Drag-and-drop `index.html` into any modern browser to play immediately.

### Deployment
- **Primary:** Project is wired for GitHub Pages via `.github/workflows/pages.yml`; enable Pages for `jordan-thirkle/neon-legends` and it will auto-deploy from `/`.
- **Hosting:** Any static host works too. Built as pure mobile-first web app.
- **App Store:** Wrap with Capacitor/Cordova using the PWA manifest + service worker already included.
- **Domain:** Assign a custom domain once deployed.
