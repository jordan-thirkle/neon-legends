# Neon Legends — Idle RPG

Drag-and-drop `index.html` into any modern browser to play immediately.

### Deploy
- **Hosting:** Drop all files into any static host. Vercel / Netlify / GitHub Pages / Firebase Hosting.
- **App Store:** Wrap with [Capacitor](https://capacitorjs.com/) or [Cordova](https://cordova.apache.org/) — the PWA manifest + service worker are already included.
- **Domain:** Assign a custom domain once deployed.

### Monetization Hooks Included
- Summon banners with pity timer
- IAP shop placeholders ($0.49 - $4.99 USD)
- Ad placement slots (can add AdMob / Unity Ads via wrapper)
- Battle pass
